SwiftShift: Arcade Driving Game

C# Unity Game Dev

Role: Game Developer

Tech Stack: C#, Unity, Visual Studio Code, TextMeshPro

Project Overview

SwiftShift is a single-player arcade driving game featuring a "Collision Avoider" mechanic with smooth lane-switching dynamics and progressively increasing difficulty over time.

Key Features

  • Smooth lane-switching mechanics with fluid transitions
  • Continuous road scrolling using texture offset to simulate forward motion
  • Realistic steering controls with adaptive turn angles based on speed
  • Dynamic obstacle spawning system with randomized traffic patterns and variable vehicle speeds
  • Clean, responsive user interface with main menu, score tracking, and game over panel
  • Utilized coroutines to manage acceleration, obstacle generation, and scoring without frame interruption
  • Optimized object lifecycle to improve performance and prevent resource overload
  • Documented and structured project for scalability, with clear pathways for future features (difficulty levels, leaderboards, multiplayer)

Technologies & Tools

Language: C#
Engine: Unity
Editor: Visual Studio Code
UI Framework: TextMeshPro
